# 7 · Gestión de riesgo (no juzgar la vela en curso)

Reglas que nacen del principio "nunca una sola vela":

1. **Espera el cierre.** En M1, no dispares sobre una vela abierta. El Close es
   el único dato estable; el High/Low de una vela abierta aún se mueve.
2. **Expiración 3 min.** Das 3 velas M1 para que el rechazo en la banda se
   confirme o se niegue. No apuestes a la vela 1 sola.
3. **El rastro invalida la entrada.** Si las velas previas (M5/M15) no dejaron el
   rastro de rango/fractal, una vela M1 bonita NO es señal. Rechazar.
4. **Mecha fugaz ≠ rechazo.** Una mecha puede ser un tick de una orden grande.
   Solo cuenta como rechazo si el CIERRE quedó del lado bueno de la banda.
5. **Indecisión = no operar.** Vela doji (cuerpo casi cero) = empate. En binarias
   el cierre puede quedar donde sea → evitar.

Con Massaniello (5 ops / 3 ITM / 60 min / PRACTICE) esto encaja: cada entrada se
decide sobre formación + rastro cerrados, no sobre una vela a medias. Menos
ruido, más contexto.
